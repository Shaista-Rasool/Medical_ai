AI Healthcare Chatbot
An intelligent machine learning-based healthcare chatbot that predicts possible diseases from user-reported symptoms and provides descriptions and precautionary advice through an interactive conversation.

Overview
This project uses a Random Forest Classifier to analyze symptoms entered in natural language. The chatbot extracts relevant symptoms, refines predictions through follow-up questions, and returns:
 Predicted disease
 Confidence score
 Disease description
 Recommended precautions



✨ Key Features
Conversational chatbot interface (CLI-based)
Smart symptom extraction:
Synonym recognition (e.g., “stomach ache” → stomach_pain)
Typo handling using fuzzy matching
Machine learning prediction using Random Forest
Confidence-based results
Dynamic follow-up questions for improved accuracy
Empathy-driven responses for better user interaction
 Tech Stack
Language: Python
Libraries: Pandas, NumPy, Scikit-learn
Techniques:
Classification (Random Forest)
Natural Language Processing (basic)
